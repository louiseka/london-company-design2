## What is the London Company Mock Website?
This project is a **mock company website** created as part of the **TechNative Digital Software Development Bootcamp** to train and showcase **core HTML and CSS skills**.

The focus of the project is on **clean structure, maintainable styling, and version control**, with a visible edit history and collaboration tracked through GitHub issues.

## Features
- Structured, semantic HTML  
- Modern CSS styling using **CSS variables**  
- Smooth **CSS transitions** for interactive elements  
- Clear project structure and version history  
- Demonstrates collaborative workflow using GitHub Issues
